VMware自动挂载共享文件夹
Created2023-01-04
VMware有个非常实用的共享文件夹功能，能够在/mnt/hgfs/路经挂载一个与主机共享文件夹，便于主机和虚拟机之间传输文件。但问题是，每次重启Ubuntu这个共享文件夹就消失了，需要重新挂载（即使设置了自动挂载也不行，具体原因未知）。这当然非常麻烦且不优雅，于是有了如下方法： 1sudo gedit /etc/fstab 在fstab文件新增一行： 1.host:/ /mnt/hgfs fuse.vmhgfs-fuse allow_other,defaults 0 0 至此，共享文件夹的自动挂载就设置好了。
我的vscode不能输入中文？
Created2023-01-04
我的vscode不能输入中文？事情是这样的，我最近的主要开发环境和博客的部署环境都用的是Ubuntu虚拟机，但是我今天突然发现我在Ubuntu下的vscode是没办法输入中文的！（就这我居然还用了好一阵子了）明明在其他软件上切到中文输入，但是一点进vscode就只能英文输入了，想要输入中文还得在其他软件写好了复制进vscode，简直不能忍！经过一番搜索发现原因是： Ubuntu的应用市场下载的code，安装的VSCode是snap的削减版本，不支持中文。 好吧，再见，麻溜的! 1sudo snap remove code 我直接去vscode官网下一个然后： 12cd ~/Downloadsudo dpkg -i code_1.74.2-1671533413_amd64.deb 再打开vscode就可以输入中文了，完美解决～
